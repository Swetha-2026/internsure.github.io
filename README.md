# InternShield AI

## Internship Verification & Risk Assessment System

InternShield AI is a web-based platform designed to help students verify internship opportunities and identify potential risks before applying. The system aims to reduce internship scams by providing risk assessment and verification support.

## Features

* Internship legitimacy verification
* Risk assessment dashboard
* User-friendly interface
* Scam awareness and prevention
* Internship information analysis
* Responsive design for desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Project Structure

```text
index.html
style.css
script.js
README.md
```

## How to Use

1. Open the website.
2. Enter internship details.
3. Submit the information for verification.
4. Review the risk assessment results.
5. Make informed decisions before applying.

## Project Goal

To create a safer internship search experience by helping students identify legitimate opportunities and avoid fraudulent offers.
